预设系统

5 套预设会保存本页普通设置和“实时语音翻译”页的所有设置。切换预设前会自动保存当前预设。

基础

输入框热键与麦克风热键会同时生效，请设置为不同按键。输入框热键用于打开/隐藏输入框；麦克风热键用于按住录音、松开识别、二次按下确认发送。

麦克风按键监听

VAD 持续监听模式

开启后用 Silero VAD 实时检测麦克风说话并自动识别，识别结果会保留「二次按键确认秒数」（默认 3 秒，重新说话直接覆盖），在此期间按下麦克风热键即翻译发送，流程与按键录音相同。保存设置后立即生效。

Google 语音识别配置

Google 语音识别使用上方语言代码，无需 API Key。

本地 GPU Whisper 配置

完全离线识别，使用与实时语音翻译相同的 faster-whisper 模型（默认 large-v3-turbo，模型已加载时不占额外显存），识别语言取上方「语音识别语言」。强制 CUDA GPU，不回退 CPU。

腾讯云语音识别配置

SecretId/SecretKey 请从腾讯云访问管理 CAM 的 API 密钥页面复制；默认使用一句话识别 SentenceRecognition，音频会按 16k wav 上传。中文普通话常用引擎为 16k_zh，其他语言请按腾讯云 ASR 文档填写。

按住热键期间录音，松开热键后识别为文字；在“二次按键确认秒数”内再次按下热键，才会继续翻译并生成 TTS。这里不是虚拟输出麦克风，可以和输出设备完全不同。

翻译与气泡

Google 翻译配置

Google 翻译使用源语言和目标语言，无需 API Key。

微软翻译配置

源语言可填 auto 自动检测；Azure Translator 多服务资源通常需要填写 Region。

腾讯翻译配置

SecretId/SecretKey 请从腾讯云访问管理 CAM 的 API 密钥页面复制，不是 AppID；源语言可填 auto 自动检测；常用中文源语言会自动把 zh-CN 转为腾讯接口使用的 zh。

本地 LLM 翻译配置

完全离线翻译，中日英互译。API 地址不带 /v1 时走 Ollama 原生接口（推荐，自动关闭思考模式速度更快）；带 /v1 时走 OpenAI 兼容接口（LM Studio / llama.cpp server 等）。推荐模型 qwen3.5:4b（速度快）或 qwen3:8b（质量略好），安装：ollama pull qwen3.5:4b。本渠道在实时语音翻译页同样可选。

百度翻译配置

App ID 和密钥请从百度翻译开放平台通用翻译 API 应用中获取；源语言可填 auto 自动检测；目标日语会自动把 ja 转为百度接口使用的 jp。

气泡格式可用变量：{original}、{translated}。透明度范围 0.1 到 1，数值越小越透明。

图片翻译

按热键截取该窗口画面（默认 VRChat，即头显视角）提交百度图片翻译，结果图显示在 VR 中；翻译中再按取消、出图后再按关闭，仅 VR 模式可用。凭证复用上方「百度翻译配置」的 App ID/密钥，并需在百度开放平台单独开通「图片翻译」。窗口关键字留空则改为截主显示器。
