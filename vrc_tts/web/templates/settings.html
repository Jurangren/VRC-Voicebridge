VRC TTS Text 设置

基础

热键配置

输入框热键与麦克风热键会同时生效，请设置为不同按键。输入框热键用于打开/隐藏输入框；麦克风热键用于按住录音、松开识别、二次按下确认发送。

麦克风按键监听

按住热键期间录音，松开热键后识别为文字；在“二次按键确认秒数”内再次按下热键，才会继续翻译并生成 TTS。选择系统默认时会直接使用操作系统默认输入设备。这里不是虚拟输出麦克风，可以和输出设备完全不同。

翻译与气泡

Google 翻译配置

Google 翻译使用源语言和目标语言，无需 API Key。

微软翻译配置

源语言可填 auto 自动检测；Azure Translator 多服务资源通常需要填写 Region。

腾讯翻译配置

SecretId/SecretKey 请从腾讯云访问管理 CAM 的 API 密钥页面复制，不是 AppID；源语言可填 auto 自动检测；常用中文源语言会自动把 zh-CN 转为腾讯接口使用的 zh。

百度翻译配置

App ID 和密钥请从百度翻译开放平台通用翻译 API 应用中获取；源语言可填 auto 自动检测；目标日语会自动把 ja 转为百度接口使用的 jp。

可用变量：{original}、{translated}。透明度范围 0.1 到 1，数值越小越透明。
